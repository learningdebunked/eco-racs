# Contributing to Carbon-Aware Checkout

Thank you for your interest in contributing to CAC!

## Development Setup

1. Fork the repository
2. Clone your fork
3. Create a virtual environment
4. Install dependencies: `make install`
5. Run tests: `make test`

## Code Style

- Follow PEP 8
- Use type hints
- Format with Black: `make format`
- Lint with flake8: `make lint`

## Testing

- Write tests for new features
- Maintain >80% code coverage
- Run full test suite before submitting PR

## Pull Request Process

1. Create a feature branch
2. Make your changes
3. Add tests
4. Update documentation
5. Run `make test` and `make lint`
6. Submit PR with clear description

## Areas for Contribution

- Additional LCA data sources
- Improved optimization algorithms
- Better behavioral models
- UI/UX for checkout interface
- Documentation improvements
- Bug fixes

## Questions?

Open an issue or contact the maintainers.
